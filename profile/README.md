## 🇬🇧 About us

**Prime Forge** is a game development and publishing team focused on building, adapting and releasing games across multiple platforms.

We work with WebGL, mobile and console projects, helping games move from technical preparation to release and post-launch support.

Our goal is to make games stable, accessible and ready for launch on international platforms.

### What we do

- Game development and project improvements
- Porting and platform adaptation
- WebGL optimization and browser build preparation
- SDK, analytics, ads and payment integrations
- Store and platform release preparation
- QA, technical audits and release support
- Marketing materials, trailers and game page preparation

### Focus areas

- Web games
- Mobile games
- Console releases
- Publishing support
- SDK documentation
- Technical review and optimization
